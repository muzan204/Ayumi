## 🌌 Sobre o projeto

**Dark Space Gothic** é um site experimental com estética sombria e espacial, desenvolvido com HTML, CSS e JavaScript.

O projeto combina:

- 🌑 Estética gótica (preto, vermelho, sombras)
- 🌌 Ambiente espacial (vazio + estrelas)
- ⚡ Efeitos glitch
- 🩸 Animações visuais de “horror UI”
- 🎸 Interface estilizada e imersiva

---

## ✨ Funcionalidades

- 🌟 Fundo com estrelas animadas
- 🩸 Efeito de sangue caindo
- ⚡ Título com glitch animado
- 🌌 Fundo com camadas espaciais
- 🎧 Botão com som interativo
- ☠ Mensagens aleatórias sombrias
- 🎨 Interface estilo glass + neon vermelho
